# Interpreter in Go

Currently supports :
- Integers
- Booleans
- Strings
    - Builtin function (len)
- Variable (dynamically typed)
- Conditionals (if else)
- First order functions
- Arrays (supports any type)
   - Builtin functions (len, first, last, tail)
- HashMaps
